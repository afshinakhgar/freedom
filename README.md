## 🚀 اجرای مستقیم اسکریپت‌ها از گیت‌هاب روی سرور

### 🔹 اجرای اسکریپت نصب WireGuard + Xray (allv2.sh)

این اسکریپت اصلی برای نصب و کانفیگ کامل تونل‌سازی و Xray است.
فقط این دستور رو روی سرور بزن:

```bash

```
